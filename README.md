**Programming for Economists Project Assignment - “Quantifying a Social Problem" (Group 2 5)**

The data used in this assignment was aquired from Eurostat. To find these datasets (and others), go to the Eurostat website and use the search bar to lookup the desired indicator. Alternatively, at the top of the Eurostat hompage there will be a section called Data. Under that section, there is a option the view the Eurostat Database where many datasets will be listed.
